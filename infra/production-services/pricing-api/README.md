These manifests were generated by the Backstage "Promote to Production" template.

## What's Included

- **namespace-claim.yaml**: Crossplane NamespaceClaim that provisions the `red-prod` namespace on aks-app-spoke
- **argocd-application.yaml**: ArgoCD Application that deploys from your existing repository

## Ingress (in separate folders)

- **istio-hub/**: VirtualService on hub cluster to route `pricing-api-prod.chrishouse.io` to spoke
- **istio-spoke/**: VirtualService on spoke cluster to route to service
